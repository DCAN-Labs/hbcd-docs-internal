## Database/Scoring

* BISQ (*ph_cg_bisq*) Score fields  
    * Added:  
        * ‘*parent_behavior_score*’  
        * ‘*parent_perception_score*’  
        * ‘*total_score*’  
    * Excluded (no data available):  
        * ‘*infant_sleep_score’*  
        * Can be calculated based on: ‘*total_score’**3 – (‘*parent_behavior_score’* + ‘*parent_perception_score’*)  
* CHAOS (*mh_cg_chaos*)  
    * Added prorated scoring  
* EEG Checklist (*eeg_ch_chkl*)  
    * Implemented mapping and correct allocation of values  
* ecPROMIS instruments - Implemented prorated scoring and rescored:  
    * ecPROMIS (1-5 y/o) - Caregiver-Child Interactions (*mh_cg_pms__cc__1to5*)  
    * ecPROMIS (<1 y/o) Caregiver-Child Relationship Scale (*mh_cg_pms__cc__inf*)  
    * ecPROMIS - Peer Relations (*mh_cg_pms__peer*)  
    * ecPROMIS - Self Regulation (*mh_cg_pms__selfreg*)  
* eHITS (*sed_bm_ehits*)  
    * Made '*total_score*' blank if all responses are NA/DK/DTA  
    * Removed *'score'* field  
* ‘Pex Health’ forms  
    * Adjustment to REDCap DD  
    * Triggered import of missing values  
* TLFB (*pex_ch_tlfb*):  
    * TLFB breakdown fields (reported frequency use for substance subcategories per week) included  
    * Modified parser to extract more accurate ‘Date of Administration’ for subset of cases submitted late (based on ‘*time_first_session*’ instead of ‘*time_completion*’)  
* 'TIC Screener' (*sed_cg_tic_screener*)  
    * Added 'Candidate Age' (*candidate_age*)  
    * Based on 'Visit Start Date' as proxy for ‘Date of Administration’
