#### MRI 
* **Data Summary Form** - Added DD Descriptions for fields '*001*' (Session No.) and '*002*' (Scan Sequence)

#### Neurocognition & Language

- **SPM2 - Missing T-Scores** - Added missing T-score fields as noted above, but pending issue to move text in score fields to separate 'notes' fields (to be done for 3.0 and documented as separate known issue)

#### Pregnancy & Exposure
* **APA** - Level-2 data removed when Level-1 criteria was not met across all sub-scales except 'Depression' subscale (Level-2 always administered and criteria removed)  

##### Physical Health
* **ecPROMIS PAGS/Sleep** - Added score fields, but pending issue for ecPROMIS-PAGS to move text in score fields to separate 'notes' fields (to be done for 3.0 and documented as separate known issue)
    * *ph_cg_pms_pags_total_score*  
    * *ph_cg_pms_pags_t_score*  
    * *ph_cg_pms_pags_se_score*

#### Social & Environmental Determinants
* **eHITS** - Updated score  
    * If all responses are DK/DTA/NA, no score is produced (Blank)  
    * 'Total score' set to 'blank' when all entries are DK/DTA/NA 
- **HOME 21 - Infant/Toddler - Scoring adjustments** - Ensured only one sub-scale is not scored if missing values on fields attributed to a single sub-scale
